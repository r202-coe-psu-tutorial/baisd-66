# Run Your Test

## Setup
Read and follow this section first time

```console
pip install nose2 coverage
```

## Directory Structure

check your directory like hear

```console
$ ls
coe_number  staircase  tests

$ tree
.
├── coe_number
│   └── number_utils.py
├── staircase
│   └── staircase.py
└── tests
    ├── test_number_utils.py
    └── test_staircase.py

```

## Run your test
Run the nodse2 command 

```console
$ node2 -v
```

or

```console
$ python3 -m node2 -v
```

run with coverage


```console
$ python3 -m node2 -v --with-coverage
```
